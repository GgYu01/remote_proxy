### [ISSUE-002] 隐匿性升级 (Stealth Upgrade)
- **问题**: 用户要求“最高隐匿性”与“省流量”。
- **分析**: 原 VMess 协议存在额外头部开销且易被探测；Trojan 需域名且 SNI 易暴露。
- **解决**: 
  1. 引入 **VLESS + Vision + Reality** 组合。
  2. 编写 `gen_keys.sh` 自动生成 Reality 密钥对。
  3. 配置 `stack: system` 优化 TCP 指纹。
- **状态**: ✅ 已解决
